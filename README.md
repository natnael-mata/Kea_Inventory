# Kea - Inventory Management System

**Project Overview:**  
Kea is an Inventory Management System designed to manage products, stock, suppliers, and sales efficiently. This project is developed using Django and Django REST Framework, with a modular app structure for scalability.

---

## **Project Structure**

- `products/` - Handles products and categories  
- `stock/` - Manages stock in/out and quantities  
- `suppliers/` - Manages supplier information  
- `sales/` - Handles sales and customer records  
- `users/` - Manages authentication and roles
